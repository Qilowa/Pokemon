Choix techniques

Nous avons utilisé les fichiers donnés pour charger le pokedex ainsi que les capacités, mais nous avons du trouver un autre document avec les statistiques de chaque pokemon.
Ces statistiques sont contenues dans la classe Fightingpokemon parce que nous ne voulions pas surcharger la classe Pokemon.
Nous avons utilisé quelques streams (la méthode toString() de Pokedex et la méthode getLength(Capacity[] c) de Print qui permet de connaître le nombre de capacité non null dans un tableau).
